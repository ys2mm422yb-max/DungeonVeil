# Complete Runtime Evidence

This focused hotfix suite records successful browser videos and screenshots for:

- Solo rooms 1–50
- Duo rooms 1–50 with a rendered teammate
- room 16/18 hazard cleanup after the final living enemy dies
- forced WebGL context loss and recovery
- World Boss motion
- normal full-game smoke motion

The runtime evidence bridge is available only on localhost and only after the test runner sets `dungeon-veil-runtime-evidence-v1` in session storage. It is not available on the published game origin.
