# Block 13 – Solo combat matrix V4

This focused block adds a deterministic 756-scenario combat matrix across seven equipment builds, six combat-relic profiles, three gift profiles and six chapter/room checkpoints.

Measured outputs include time to kill, required hits, outgoing and incoming damage, healing pressure, attack cooldown, projectile load, mitigation, room duration and a bounded mobile-control score.

The focused audit is wired into the final, progression, economy, Duo-balance and room audit suites. No Supabase schema, social flow, Duo authority or `main` branch is changed by this block.
