# Mobile room navigation acceptance

- Top-wall portals render at the same in-room stage used by tiles, lights and collider clearance.
- Banners and heraldry are wall-mounted above the floor and never create collision.
- Enemies choose a free collider-corner waypoint before using angular recovery.
- Room 2 command table: enemies can approach from either side without jogging into the table.
- Room 3 colonnade: enemies preserve forward progress through all three lanes.
