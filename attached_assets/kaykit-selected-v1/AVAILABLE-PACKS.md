# Available KayKit source packages

This file records which complete KayKit packages were available in the private Replit workspace when the curated public runtime selection was prepared.

The full source/download packages are **not** committed to this public repository. Only selected runtime assets, direct dependencies, textures and license files belong in the tracked game tree.

## Private runtime staging prepared

A private runtime staging folder was prepared from those packages containing web-usable formats and supporting files such as:

- GLB
- GLTF
- BIN dependencies
- PNG/JPG textures
- license and readme files

At the time of preparation, that private staging contained approximately 1,532 files and used approximately 117 MB.

## Public repository policy

Do not publish complete ZIP archives, Blender source files, FBX/OBJ exports, Unity packages or complete package mirrors merely for storage. When another model is needed, copy only the specific runtime model and all of its direct dependencies, textures and relevant license file into a focused asset branch.

## Backup note

The Replit workspace should not be treated as the only permanent backup of purchased or downloaded packages. Keep the original downloads privately backed up outside the public GitHub repository so they remain available even if the workspace is reset or removed.
