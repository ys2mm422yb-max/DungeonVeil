# Dungeon Veil - Room Bible Chapter 1

This branch contains the binding design concept for rooms 1-20.

The concept defines:
- four visual phases across Chapter 1
- six distinct combat silhouettes
- one hero object per room
- allowed and forbidden asset families
- room-specific light and VFX direction
- enemy formations and portal staging
- mobile camera safe-zones
- an equipment-preview specification for crossbows

The full 26-page room bible is maintained in Canva and used as the source of truth before runtime implementation.
